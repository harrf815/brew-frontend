BREW PROJECT :::
----------------------
Do you love beer?! obviously! Brew Project is here for you... search, rate, and comment on any brewery from across the United States!


Development set up :::
----------------------
fork and clone Backend Brew 
bundle install
run rails server

fork and clone frontend Brew
npm install
run npm start

Technologies :::
----------------------
